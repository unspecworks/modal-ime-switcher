# Change Log

All notable changes to the "modal-ime-switcher" extension will be documented in this file.

## [0.0.1] - 2026-01-15

### Added

- **Smart IME Switching**: Automatically switches to configured IME (e.g., English) when entering Normal Mode.
- **Auto Restore**: Saves used IME (e.g. Korean) when leaving Insert Mode and restores it upon return.
- **Window Focus Integration**: Restores IME when window loses focus; enforces Normal IME when window gains focus in Normal Mode.
- Configuration for `im-select` binary path and Normal Mode cursor definitions.
